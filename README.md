# Sextant FX Dashboard

Front-end only live FX decision dashboard for SGD → IDR using a zone-based rule engine (Hold / Convert / Accelerate).  
Built for the Sextant Protocol “Christine Trust” long-term currency allocation framework.  
No backend, fully static, GitHub Pages deployable.

## 📊 Decision Zones

| Zone | Range | Action |
|------|-------|--------|
| A | 13,800 – 14,200 | Hold / Small Convert |
| B | 14,200 – 14,600 | Convert 10–20% |
| C | 14,600 – 15,000 | Aggressive Convert |
| D | 15,000+ | Exit Remaining Position |
